# 📌 CampusBoard

A community noticeboard web app where users can post and delete notices.

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Storage: In-memory (array)

## ✨ Features
- Post a notice with title, description and author name
- View all posted notices
- Delete any notice

## 🚀 How to Run

### Backend
cd backend
node server.js

### Frontend
cd frontend
npm start
